- Role check: ADMIN | PRINCIPAL
- Read `formData.get("rows")` — this is a JSON string of validated rows
- Parse JSON: `const rows = JSON.parse(formData.get("rows") as string) as Array<{ lrn: string, lastName: string, firstName: string, middleName: string, gradeLevel: string, sectionName: string, strand: string, sex: string, contactNumber: string }>`
- Validate with Zod: each row through a schema similar to createStudent
- Resolve sections: For each row, look up the Section by matching gradeLevel -> GradeLevelEntry (in active school year) -> sections.find(s => s.name === row.sectionName). For SHS (G11/G12), also match strand on the section.
- Use `prisma.$transaction` (callback style) to:
  1. For each valid row: create Student, then auto-enroll (same logic as createStudent — find SubjectAssignments for section's gradeLevelEntryId + strandId, createMany StudentEnrollment)
  2. Skip rows where LRN already exists (check with `tx.student.findUnique({ where: { lrn } })`) — track skipped count
  3. Skip rows where section not found — track skipped count
- Return `{ success: true, imported: successCount, skipped: skipCount }`
- `revalidatePath("/dashboard/enrollment")`

**IMPORTANT: Handle errors gracefully.** If a single row fails (e.g., duplicate LRN), skip it and continue — do NOT abort the entire transaction for one bad row. Use try/catch inside the loop within the transaction.

**`parseCSV` function (inside component file or separate utility):**
```typescript
type ParsedRow = {
  lrn: string
  lastName: string
  firstName: string
  middleName: string
  gradeLevel: string
  sectionName: string
  strand: string
  sex: string
  contactNumber: string
}
type ParseError = {
  rowIndex: number
  errors: string[]
  raw: string
}

async function parseCSV(file: File): Promise<{ valid: ParsedRow[], errors: ParseError[] }> {
  const text = await file.text()
  const lines = text.split("\n").map((l) => l.trim()).filter(Boolean)
  const [_header, ...rows] = lines
  const valid: ParsedRow[] = []
  const errors: ParseError[] = []

  rows.forEach((line, idx) => {
    const cols = line.split(",").map((v) => v.trim())
    const [lrn, lastName, firstName, middleName, gradeLevel, sectionName, strand, sex, contactNumber] = cols

    const rowErrors: string[] = []
    if (!lrn) rowErrors.push("LRN required")
    if (!lastName) rowErrors.push("Last name required")
    if (!firstName) rowErrors.push("First name required")
    if (!["G7", "G8", "G9", "G10", "G11", "G12"].includes(gradeLevel)) rowErrors.push("Invalid grade level")
    if (!["Male", "Female"].includes(sex)) rowErrors.push("Sex must be Male or Female")
    if (["G11", "G12"].includes(gradeLevel) && !strand) rowErrors.push("Strand required for SHS")

    if (rowErrors.length > 0) {
      errors.push({ rowIndex: idx + 2, errors: rowErrors, raw: line })
    } else {
      valid.push({ lrn, lastName, firstName, middleName: middleName || "", gradeLevel, sectionName, strand: strand || "", sex, contactNumber: contactNumber || "" })
    }
  })

  return { valid, errors }
}
```
